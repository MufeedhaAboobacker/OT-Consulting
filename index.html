510(k) Filing

Each medical device in the US, classified as Class II, requires the manufacturer to register its site, their products, and have a quality system in place. OTech assists with filing the FDA documents (510(k)) to get clearance for selling the product.

Quality System Implementation

The implementation of a quality system is a requirement for any manufacturer of medical devices in the US, regardless of its classification, i.e. whether it needs a 510(k) filing or not. OTech has assisted with the implementation of these quality systems, ranging from documenting existing processes and procedures, performing audits for compliance, to implementing a system from scratch. We have templates that can be used by vendors to be customized for the system implementation.
